Judul : Manajemen Keuangan Keseharian berbasis Web dan Mobile menggunakan Flutter dan Dart

Nama Aplikasi : DompetKu

Latar Belakang :
disini saya membuat aplikasi web mobile tentang manajemen keuangan keseharian karena saya melihat dari permasalahan saya sendiri dimana untuk menagatur keuangan itu hal yang belum bisa saya maksimalkan, entah dari keperluan sehari hari maupun keperluan darurat, dengan aplikasi web ini saya harap bisa mengatasi hal itu

Fitur - Fitur :
1. Transaksi =
   fitur ini dibuat untuk menambah transaksi seperti pengeluaran ataupun pendapatan tambahan

2. Budget =
   fitur ini diperuntukkan ketika ingin budgetting perminggu ataupun perbulan. agar pengguna tau batasan ketika melakukan transaksi terutama pada bagian pengeluaran

3. Kategori =
   fitur ini hanya untuk menambah kategori di pemasukan dan pengeluaran

4. Profile =
   fitur profile bisa untuk melihat profil dari user yang login dan juga untuk logout dari akun
